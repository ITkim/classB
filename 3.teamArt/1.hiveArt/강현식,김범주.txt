1. hive에서 table 생성시
  - hive에 파일저장"순서"에 관계없이 한글>영어 순으로 저장되며
    오름차순으로 파일이 저장되므로 테이블도 파일 오름차순으로 생성된다.
    -> 따라서 파일명에 신경을 써야한다!
    -> 한 파일에 대해서만 테이블을 생성하고 싶다면 파일마다 상위폴더를 생성해주어야 한다.

  - csv, txt 모두 테이블 생성은 되지만 생성 시 txt파일 특성상 ',' 로 구분 되지 않음
  - 테이블 생성 시 선언한 변수들의 자료형과 파일 내에서의 변수들의 자료형을 일치 시켜주지 않으면
    NULL값으로 표시된다.

2. hive table 삭제시
  - create 'external' 을 해준다면 drop 시에도 hive 하위폴더에 파일들이 유지가 된다.
  - create 'external' 을 하지 않는다면 drop 시 전체 파일이 삭제되므로 주의해야 함
